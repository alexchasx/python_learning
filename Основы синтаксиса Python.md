## Типы данных
1. Числовые.
    В Python есть три основных числовых типа: 
    - Целые числа (int)

## КОРТЕЖ (tuple)

Кортеж — это упорядоченная и НЕИЗМЕНЯЕМАЯ коллекция элементов, которые могут быть разных типов. 
Меньше занимают памяти по сравнению со списками

Преимущества кортежей по сравнению со списками:
- меньше места занимают
- не сможете уничтожить элементы по ошибке
- можно использовать в качестве словарных ключей
- именованные кортежи могут служить более простой альтернативой объектам

"включений" кортежей не существует

СОЗДАНИЕ КОРТЕЖА:

    @name.setter
    def name(self, input_name):
        print('setter')
        self.hidden_name
